# Portfolio Development Tasks

Este documento rastrea las tareas de modernización, rediseño y nuevas características para el portfolio de Andy Gomez (Full-Stack AI Developer).

## 🎨 Fase 1: Estética y "Glow Up" Tecnológico
- [ ] **Glassmorphism Avanzado**: Cambiar los fondos sólidos de las tarjetas por `bg-black/40 backdrop-blur-xl border-white/10`.
- [ ] **Efectos Neón/Glow**: Añadir luces radiales de fondo (cyan y azul) detrás del Hero y otras secciones clave.
- [ ] **Tipografía Tech**: Integrar fuentes monospaciadas (ej. JetBrains Mono) para elementos de código y etiquetas, manteniendo una fuente sans limpia para legibilidad.
- [ ] **Micro-interacciones**: Efectos de "hover" magnéticos en las tarjetas de Skills y Projects.

## 👤 Fase 2: Personalización y Autoridad
- [ ] **Integración de Foto/Icono**: Adaptar el Hero u otra sección prominente para incluir la foto de perfil o logo proporcionado por el usuario, integrándolo estilísticamente con el entorno (ej. estilo holográfico o integrado en el bloque de código).
- [ ] **Sección de Certificaciones**: Crear una sección nueva bajo Skills o Projects dedicada a mostrar medallas/certificados de IA o Cloud (AWS, GCP, etc.).

## 🚀 Fase 3: Dinamismo en Proyectos
- [ ] **Integrar Imágenes Reales**: Reemplazar los placeholders (iconos de cohete) por las imágenes reales de los proyectos proporcionadas por el usuario.
- [ ] **Animaciones Scroll/Parallax**: Implementar efectos Parallax para que las imágenes de los proyectos se muevan o cobren vida conforme el usuario hace scroll.

## 📧 Fase 4: Newsletter & Comunidad
- [ ] **Sección de Suscripción**: Crear un bloque de "Newsletter" (ej. "La Trinchera del Desarrollador AI") para captar correos electrónicos.
- [ ] **Integración de API**: Conectar el formulario de la newsletter con la API en Python (u otro servicio como Resend/Mailchimp) para registrar correctamente los correos.
